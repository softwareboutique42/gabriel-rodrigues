# Requirements — Company Canvas (v1.7 Casinocraftz Foundation and Slots Integration)

**Defined:** 2026-04-03
**Core Value:** Branded, download-ready animation in under a minute.

## v1.7 Requirements

### Casinocraftz Foundation

- [ ] **CCZ-40**: Canonical EN/PT browser routes introduce Casinocraftz as a transparent, zero-risk casino simulation with explicit anti-gambling framing.
- [ ] **CCZ-41**: The Casinocraftz shell can host the first playable module cleanly and feels distinct from the existing Canvas and standalone Slots surfaces.

### Slots Integration and Improvement

- [ ] **SLOT-40**: The Slots game runs inside Casinocraftz as the first playable module without losing deterministic authority, parity, or lifecycle safety.
- [ ] **SLOT-41**: The slot experience is improved where it helps the educational loop, especially around clarity of odds, outcomes, blocked states, and manipulation cues.

### Tutorial, Dialogue, and Progression

- [ ] **EDU-40**: The first-run tutorial focuses on house edge as the primary early lesson.
- [ ] **EDU-41**: Educational delivery combines narrative dialogue with direct real-time system explanations.
- [ ] **SYS-40**: AI Essence and three starter utility cards, Probability Seer, Dopamine Dampener, and House Edge, provide the first progression loop.
- [ ] **SYS-41**: Tutorial events, dialogue, and card utilities interact deterministically rather than as disconnected features.

### Validation and Release Confidence

- [ ] **SYS-42**: The integrated Casinocraftz experience preserves a fake, free-to-play economy with no real-money or microtransaction drift.
- [ ] **QA-40**: Contract and compatibility coverage protect canonical EN/PT Casinocraftz routes, embedded Slots behavior, tutorial flow, and starter-card interactions.
- [ ] **QA-41**: Release verification captures lint, targeted tests, browser checks, and build evidence for the integrated POC milestone.

## v2+ Requirements (Deferred)

### Future Expansion

- **SYS-50**: PvP or collection-driven card battling beyond utility-card education.
- **EDU-50**: Additional psychology lessons beyond house edge, including near-miss manipulation and sensory conditioning.

## Out of Scope

| Feature                              | Reason                                                                        |
| ------------------------------------ | ----------------------------------------------------------------------------- |
| Real-money wagering or deposits      | The project is explicitly educational and zero-risk.                          |
| Microtransactions or paid card packs | The prototype should not recreate the monetization pressure it is critiquing. |
| PvP battles in the first milestone   | Starter cards are utility tools first, not a full combat meta.                |
| Native mobile or Discord clients     | The first milestone is browser-first for accessibility and speed.             |
