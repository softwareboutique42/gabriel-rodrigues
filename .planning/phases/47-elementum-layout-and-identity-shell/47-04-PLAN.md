    Grid gap: `clamp(0.6rem, 1.4vw, 1.2rem)`

    - 2.2rem = 35.2px (sm minimum)
    - 5vh (varies; on 800px: 40px)
    - 3rem = 48px (lg maximum)
    - Smooth scaling: 35.2px → 48px ✓
    
    Spin button: `clamp(5rem, 12vh, 7rem)` (Same as HUD row ✓)
    
    All clamp() values follow the pattern: min ≤ middle ≤ max ✓
