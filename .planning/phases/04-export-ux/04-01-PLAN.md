# Phase 4 Plan 01: Export Settings Contracts and Progress Baseline

## Objective

Introduce export settings contracts (format/aspect ratio/quality), map them to concrete capture dimensions, and expose a visible progress bar baseline in EN/PT.

## Implemented Tasks

1. Added export settings types and defaults for format, aspect ratio, and quality.
2. Added profile resolution for 16:9, 1:1, and 9:16 in 1080p and 720p.
3. Updated the export controller to use resolved profile dimensions and frame totals.
4. Added progress bar markup to both canvas pages and wired progress updates in main flow.
5. Added bilingual progress-label template strings.
6. Extended export contract tests for FR-4.1 and FR-4.4.
