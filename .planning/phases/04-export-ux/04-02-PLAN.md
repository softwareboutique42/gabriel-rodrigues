# Phase 4 Plan 02: Export Modal and Option Controls

## Objective

Build the export modal shell and user controls for format, aspect ratio, and quality selection in EN/PT.

## Planned Tasks

1. Add modal container and trigger flow from the canvas download action.
2. Add format selector (WebM/MP4 with capability-aware availability states).
3. Add aspect ratio selector (16:9, 1:1, 9:16).
4. Add quality selector (1080p/720p).
5. Persist selected options into `startVideoExport` call settings.
6. Add EN/PT strings for all modal labels and helper text.
